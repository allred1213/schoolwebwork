<!DOCTYPE>


This element is used to tell the browser what mode in which to render.
You should always include this tag in an HTML document.

Example:
<!DOCTYPE html>
<html lang="en">
<head>
</head>
</html>

<a>


This is an anchor element.
It provides the means to create a hyperlink to another page.
It can also serve as an anchor within a page to which a hyperlink points to.

Example:
<a href="http://www.androidcentral.com">Android Central"</a>

Result:
Android Central

<abbr>

This element is used to mark up abbreviations.
You can use the title attribute to add accessibility features.

Example:
<p>The <abbr title="Graphical User Interface">GUI</abbr> of the company's app is long overdue for a re-design.</p>

Result:

The GUI of the company's app is long overdue for a re-design.

<area>

This element is used to define an area inside of an image with clickable areas, or an image-map.

Example:
<p>Click on a building to see it up close</p> <img src="images/nyc.jpg" width="200" height="200" alt="New York City" usemap="#nyc"> <map name="planetmap"> <area shape="rect" coords="0,90,40,70" alt="Sun" href="images/esb.jpg"> </map>

Results:

Click on the Empire State building to see another picture

<article>

This tag is used to specify a self-contained area of content.
For example - a news article, a movie review, or a blog entry.

Example:

Results:

As always, there were many apps to choose from this round up.
So, let's get started with the first one.

The aside element indicates a section of content that is tangentially related

Results:

<bdo>

bdo stands for Bi-Directional Override. It is used to override the current text direction.

Example
<p>The next paragraph will flow right from left.</p>
<p><bdo dir="rtl">The next paragraph will flow right from left.</bdo></p>

Results

The next paragraph will flow right from left.

The next paragraph will flow right from left.
